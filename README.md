# YUVStreamReciever
A simple raw video stream reciever via TCP/UDP protocol.
**Requires OpenCV lib.**

- Для просмотра стрима на ПК необходимо запустить catch_stream.sh
- Для запуска стрима и работы с геймпадом с Raspberry необходимо запустить start.sh, который уже лежит в домашней папке pi, и программу GamepadTCPClient на ПК.
